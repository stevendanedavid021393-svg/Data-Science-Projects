Linear Regression — Step-by-Step Math

Using the least squares method, we find the line ŷ = bx + a that best fits the data, where:

  • b (slope) = (Σxy − n·x̄·ȳ) / (Σx² − n·x̄²)
  • a (y-intercept) = ȳ − b·x̄

Step 1 — Data Table

x (min)y (lbs)xyx²
Step 2 — Compute Means n = 8
x̄ = Σx / n = 505 / 8 = 63.125
ȳ = Σy / n = 1426 / 8 = 178.25
Step 3 — Compute Slope (b) b = (Σxy − n·x̄·ȳ) / (Σx² − n·x̄²)
b = (90,061 − 8 × 63.125 × 178.25) / (32,933 − 8 × 63.125²)
b = (90,061 − 90,016.25) / (32,933 − 31,878.125)
b = 44.75 / 1,054.875
b ≈ 0.042
Step 4 — Compute Y-Intercept (a) a = ȳ − b·x̄
a = 178.25 − (0.042 × 63.125)
a = 178.25 − 2.651
a ≈ 175.57
Regression Line:   ŷ = 0.042x + 175.57

Interpretation: For each additional minute of running per week, weight increases by approximately 0.042 lbs. The very small slope and tight weight range (176–180 lbs) suggest a weak positive correlation in this dataset — other variables likely have a stronger influence on weight.
