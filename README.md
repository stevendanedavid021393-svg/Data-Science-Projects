# Data Science Projects

Welcome to my Data Science Projects repository! This repository serves as a portfolio of work completed throughout my data science coursework and personal learning journey.

## About This Repository

Here you will find a collection of projects covering core data science topics, including:

- **Data Wrangling & Exploration** — cleaning, transforming, and exploring real-world datasets
- **Statistical Analysis** — descriptive statistics, hypothesis testing, and probability
- **Data Visualization** — charts, graphs, and dashboards using tools like Matplotlib, Seaborn, and Plotly
- **Machine Learning** — supervised and unsupervised models (regression, classification, clustering)
- **Natural Language Processing** — text analysis and feature extraction
- **Python Programming** — scripts, notebooks, and utilities built with Python and Pandas

## Tools & Technologies

- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebooks
- SQL
- Data visualization libraries

This repository is actively maintained as part of my coursework and serves as an evolving portfolio that demonstrates my growth and skills as a data scientist.
